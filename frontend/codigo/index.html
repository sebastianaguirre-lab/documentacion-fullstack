CSS extra

Prioridad CSS

Cuando varias reglas afectan al mismo elemento, CSS revisa la prioridad.

  1. ID: mayor prioridad.
  2. Clase: prioridad media.
  3. Etiqueta: prioridad menor.
/* Etiqueta */
p {
    color: blue;
}

/* Clase */
.texto {
    color: green;
}

/* ID */
#mensaje {
    color: red;
}

Cascada CSS

Si dos reglas tienen la misma prioridad, gana la ultima escrita.

.caja {
    background-color: yellow;
}

.caja {
    background-color: orange;
}

JavaScript aplicado - Desafío Notas Finales

Esta seccion muestra el flujo completo entre JavaScript y HTML usando variables, prompt, conversion de datos, calculos y cambio de contenido en la pagina.

1. Variables con let

let permite guardar datos para usarlos despues.

// Variable de texto
let nombre = "Sebastián";

// Variable numerica
let edad = 31;

2. Entrada de datos

prompt() muestra una ventana para pedir informacion. Lo que devuelve siempre es texto.

// prompt devuelve texto
let nombre = prompt("Ingrese su nombre");

3. Conversion de datos

parseFloat() convierte texto a numero decimal. Se usa para calcular notas.

// Convertimos el texto ingresado a numero
let nota = parseFloat(prompt("Ingrese una nota"));

4. Operadores matematicos

JavaScript permite sumar, restar, multiplicar y dividir valores numericos.

let suma = n1 + n2;
let resta = n1 - n2;
let multiplicacion = n1 * n2;
let division = n1 / n2;

// Promedio de tres notas
let promedio = (n1 + n2 + n3) / 3;

5. Buscar elementos HTML

document.getElementById() permite conectar JavaScript con un elemento HTML que tenga un id.

// Buscamos una celda por su id
let celda = document.getElementById("resultado");

6. Modificar contenido HTML

innerHTML cambia el contenido visible de un elemento HTML.

// Mostramos el promedio en la pagina
celda.innerHTML = promedio;
